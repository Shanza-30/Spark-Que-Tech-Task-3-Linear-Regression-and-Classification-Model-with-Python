# Linear Regression & Classification Model

## Project Overview
This project demonstrates how to build machine learning models for both regression and classification tasks using Python to predict numeric outcomes and classify categories from real-world data. Task 3 implements Spark Que Tech for efficient data processing, including cleaning, feature engineering, and scaling, and trains scalable ML models using Spark MLlib for both regression and classification.

## Tools & Technologies
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Objectives
- Build a Linear Regression model to predict numeric values.
- Build classification models using Logistic Regression and Decision Tree.
- Evaluate models using different performance metrics.
- Create visual analytics for better understanding of the data.

## Dataset
Heart Disease Dataset from Kaggle.

The dataset includes medical attributes such as age, cholesterol level, blood pressure, and heart disease diagnosis.

## Project Steps

1. Import required Python libraries.
2. Load the dataset using Pandas.
3. Explore the dataset using info and descriptive statistics.
4. Check for missing values.
5. Visualize relationships using a correlation heatmap.
6. Prepare data for regression modeling.
7. Split dataset into training and testing sets.
8. Train Linear Regression model.
9. Predict values using the trained model.
10. Evaluate regression model using R² Score and RMSE.
11. Visualize regression predictions.
12. Prepare dataset for classification.
13. Split dataset for classification training and testing.
14. Train Logistic Regression model.
15. Evaluate classification model using accuracy, precision, and recall.
16. Train Decision Tree classifier.
17. Visualize class distribution.
18. Save the trained model.

## Evaluation Metrics
Regression:
- R² Score
- RMSE

Classification:
- Accuracy
- Precision
- Recall

## Results
The project successfully demonstrates regression and classification models along with evaluation metrics and visualizations.
